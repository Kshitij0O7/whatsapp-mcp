# Support & Supported Devices

## Recommended browser
- **Google Chrome** (use the latest version)

## Supported devices
| Device | Supported? |
|---|---|
| Desktop | ✅ Yes |
| Laptop | ✅ Yes |
| Mobile phone | ❌ Not currently supported |
| Tablet / iPad | ❌ Not currently supported |

If a customer reports problems on a phone or tablet, or on another browser, recommend switching to
the **latest Google Chrome on a desktop or laptop**.

## Common troubleshooting
- **Display / layout issues** → use the latest **Google Chrome** on desktop or laptop.
- **Can't log in / forgot password / login errors** → guide them to the login/registration flow on
  **https://solartechy.in**. If the issue persists and cannot be resolved from the knowledge here,
  raise an escalation (category `BUG` or `SUPPORT`) so the team can help.

## Getting started reminder
1. Visit **https://solartechy.in**
2. Click **Start Free Trial**
3. Register with login details + basic personal and company information
4. Access the dashboard

## When to escalate
Escalate to a human (see `escalation.md`) for:
- A **software bug** or technical issue that cannot be resolved with the guidance above
- Account issues you cannot resolve (e.g. persistent login failures)
- Anything not covered in the knowledge base
